# Release History

## 0.6.0

The repository's [`0.6.0` changelog](https://github.com/Adventurecraft-Awakening/AC-Legacy-Mod/blob/main/changelog/0.6.0.md) documents renderer, arrow, lighting, chunk-saving, and biome performance work; scripting API additions; game rules; quality-of-life improvements; and bug fixes.

The version in `gradle.properties` is the configured artifact version. Development commits after a release tag may retain that value, so use the in-game commit/version metadata or a build's Git revision when reporting a development-build bug.

The Fandom “Version History” page primarily records historical original-mod revisions. It is preserved through [[Fandom Baseline Index]] rather than merged into the current release chronology.
